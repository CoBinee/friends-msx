## 説明

## 操作
- `←`, `→`<br>移動
- `SPACE`<br>ジャンプ
- `↑` + `SPACE`<br>一段上にジャンプ
- `↓`<br>落下する際に押すと急降下
- ジョイスティックにも対応

## 動作環境
- 32KB 以上の RAM を搭載した MSX
